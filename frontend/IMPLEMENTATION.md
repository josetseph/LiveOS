## 🚀 Running the Application

1. **Start Backend** (in `/backend`):
   ```bash
   uvicorn app.main:app --reload
   ```

2. **Start Frontend v2** (in `/frontend`):
   ```bash
   npm run dev
   ```

3. **Access**:
   - Frontend: http://localhost:3000
   - Backend API: http://localhost:8000
